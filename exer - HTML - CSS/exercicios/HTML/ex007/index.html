Formatação de texto

Negrito / Destaque

* Para que possamos deixar um trecho de um texto em Negrito usamos a tag B (Não semântica).

* Para que possamos deixar um trecho de um texto em Destaque, usamos a tag STRONG (Semântica).

-Semântica / Não Semântica

Veja que são tags diferentes mas com "mesmo resultado". Como dito antes as tags semântica são aquelas que tem significado não apenas forma, a tag b por exemplo ela muda a forma de um determinado trecho para deixar em Negrito, mas a tag Strong é para deixar aquele trecho em destaque.

Ênfase / Itálico

* Para que possamos deixar um trecho de um texto em itálico, nos usamos a tag I (Não semântica).

* Para que possamos deixar um trecho de um texto em ênfase, nos usamos a tag EM (Semântica).

Marcação / mudar cor da marcação

* Para que possamos fazer uma marcação num trecho de um texto, nos usamos a tag MARK (Semântica).

* Mas tambem podemos motificar a coloração dessa marcação com a tag STYLE e BACKGROUND-COLOR

* Como nesse trecho eu mudei a cor dele com essa tag que acabei de demonstrar.

* Porem você nem percebeu que na verdade eu usei outra forma de modificar a cor da marcação, eu usei a tag apresentada na imagem abaixo, na area head.

Diminuir / Almentar

* Para almentar alguns trecho usamos a tag big, porem ela já é considerada uma tag Obsoleta uma tag morta.

-Tags Obsoletas

A.. Mas a tag é obsoleta mas ela está funcionando?!
Calma para isso tem uma explicação, ela está como obsoleta mas não parou de funcionar de vez mas tambem isso não significa que ela vai durar muito tempo. então sempre é bom ficar atendo com atualizações de tags.

* E caso queira diminuir é só usar a tag SMALL essa não está obsoleta.

Deletar / Incluir

* Tambem temos tags de Inclusão e Exclusão, e nesse caso tambem temos tags semânticas e não semânticas, mas nesse caso só na tag de Inclusão (U / INS).

* Como nessa parte retirada de um trecho Lorem ipsum dolor eu usei a tag DEL.

* Mas caso eu queira inserir um novo trecho eu uso a tag INS. Mas nesse caso é uma tag semântica.
Agora se eu quiser usar uma tag não semântica eu uso o U. tem a mesma forma porem o INS ele tem significado e o U não tem.

Sobrescrito / Subscrito

* Para se fazer um texto com algumas partes Sobrescrito como: x20+3 eu uso a tag SUP

* Agora se eu quiser fazer um texto com uma parte de alguam palavar Subscrito eu uso a tag SUB como: H2O.
