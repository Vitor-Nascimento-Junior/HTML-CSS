Reproduzir Audios

aqui vamos aprender sobre colocar audios em nossos sites, veja que facil: 

        <aduio src="arquivo.mp3" controls autoplay loop></audio>
Mas caso o arquivo não rode, isso pode ser porque o navegador não suporta mp3, então você pode usar outros tipos de midias (MP3, WAV e OGG) lembrando o WAV ele é um arquivo pesado!

Tags para diferentes tipos de midia

    
        <audio preload="auto" autoplay controls loop>
        < source src="arquivo.mp3"></audio>
