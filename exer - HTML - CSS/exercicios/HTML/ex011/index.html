Imagens dinâmicas

Aqui vamos aprender um pouco sobre imagens dinâmicas, nos criamos algumas imagens no GIMP e vamos usar aqui como exemplo.

Pelo que podemos ver aqui, nos conseguimos almentar e diminuir o tamanho da imagem só de, almentar e diminuir o tamanho do site por isso imagem dinâmica.

Tudo isso usando a seguinte tag 

    <picture>
        <source media="(max-width: 750px)" type="image/png">
        <source media="(max-width: 1000px)" alt="image/png">
        <img src="imagem.png" alt="foto.png">
    </picture>
