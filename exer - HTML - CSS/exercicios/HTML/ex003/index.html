Veja como é feito para incluir imagens em sites

Você verá agora como é facil incluir imagens (png, jpeg e entre outros) de muitas formas e de maneira facil e simples

como toda ação que fazemos em nossos sites em HTML nos usamos tags. nesse caso vamos usar a tag <imag>

Nesse caso eu usei a tag <img src="" alt="">

Nessa tag temos tres siglas chaves, "img" para definir que o que vou fazer é impor uma imagem, "src" que serve para busca da imagem que desejar e "alt" onde vamos descrever o que seria aquela imagem por exemplo

<img src="logo css-full.png" alt="logo css"> dessa forma coloquei o arquivo da imagem e depois descrevi o que ela é.


Mas tambem temo como eu colocar imagens que estão dentro de pastas como essa.

Como nesse exemplo acima eu apertei ctrl+espaço é apareceu essas opcões de arquivos de imagens que estão na mesma pasta do arquivo desse meu site.
