Criando links externos

Aqui vamos aprender um pouco sobre links internos e externos, mas primeiramente vamos para o externo

Links externos / abrindo outra página

Um grande exemplo de link externo são esses.

  • acesse meu instagram
  • veja uns dos meus canais favoritos de rap geek
Como podemos ver temos dois links, um que vai para o meu perfil do instagram e outro que vai para um dos meus canais favoritos de rap geek. Mas deu para notar que assim que clicamos neles eles criam uma nova página, mas isso só é possivel por conta desses carinhas aqui target="_blank" e rel="external"

Links externos / abrindo na mesma página

Bom esse exemplo é o mesmo vamos usar a tag A para fazer a ancoragem de um site para outro. Nesse caso usei meu perfil do instagram denovo como exemplo. Mas como é essa tag inteira?
<a> href="URL"</a>
Agora se quiser fazer o site abrir em outra página sem sobrepor seu site é só usar essas tags <a> href="URL" target="" rel=""</a>

Essas tag é interessante usar também dessa forma https://www.youtube.com/@MHRAPOFICIAL usar o próprio link assim ficando até mais aparente.

Uma curiosidade sobre links

  • Nos sabemos que com a tag A podemos ancorar links um atras do outro certo. Mas depois que vimos como o google funciona, temos que fazer um esforço para que ele possa localizar nossos sites e recomende-o para terceiros. Assim vamos focar nos REL
  • REL:
    • Os rel são relações, são praticamente uma explicação do que seria exatamente aquela informação. por exemplo, quando nos falamos que <a> href="URL" target="_blank" rel="external"</a> nesse caso a tag está referindo que esse link é external (externo).
    Mas tambem temos difersos exemplo, como pode ser um link que eu não quero que ele seja seguido, não te tanta importancia, como esse: acesse a Hostnet Nesse caso eu fiz isso: <a> href="https://www.hostnet.com.br" target="_blank" rel="nofollow</a>
  • TARGET:
    • Já o target vai definir se vai mudar ou permanecer na mesma janela, como por exemplo target="_blank" essa tag vai mandar você para o link em outra aba. Agora se quiser que ele sobreponha seu site é só não colocar nada ou usar a tag target="_self"
