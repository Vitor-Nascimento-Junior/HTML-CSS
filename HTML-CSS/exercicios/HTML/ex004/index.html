olha o favicon aqui emcima

para impor um icone em seu site é muito simples, essa é a nossa primeira tag na area head, sendo assim a tag Link. Simples assim que você criar seu site na area head você vai escrever emcima do title a tag<link rel="shortcut icon" href= "favicon.icon" type="favicon.icon">.
Lendo dessa forma parece até dificil de decorar esse comando mas é so digitar link:favicon e apertar enter que ele cria o codigo, depois só colocar o icone que deseja.
