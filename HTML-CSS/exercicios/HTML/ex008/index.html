Código-fonte / Pré-formatação

Aqui vamos aprender a colocar um trecho de uma frase ou até mesmo um exemplo de código em html, com o mesmo tamanho de fonte. Veja isso: 

            
    <style>
    mark {
        background-color: rgb(155, 202, 46)
    }
    </style>
Nesse caso usei duas tags a PRE e a CODE. A pre é utilizada para dar as quebras de linhas sem precisar usar varios BR, e a tag code seria para dar o mesmo tamanho para todas as letras, assim tudo ficando mais legível.
