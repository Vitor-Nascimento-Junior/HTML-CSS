Mas como fazer isso?

Simples para que isso ocorra você teve fazer um texto no meio de duas Tags ( <p> e </p> )

Mas se eu quiser quebrar a linha do meu paragravo é só apertar Enter?

Negativo, para que uma quebra de linha seja realmente realizada é preciso que você use a tag ( <br> ) ele sim vai quebrar seu texto

Por exemplo:

Aqui É Um Texto Feito Só Apertando ( Enter ).

Já aqui é um texto
separado pela tag ( <br> )
